■ 代码块展示

javascript 
function createDarkTheme(name) {
  return {
    name,
    colors: {
      primary: '#10B981',
      codeBg: '#0F172A'
    },
    apply() {
      console.log(`Theme "${this.name}" applied.`);
    }
  };
}

const theme = createDarkTheme('dark');
theme.apply();
python 
class DarkTheme:
    def __init__(self):
        self.colors = {
            "primary": "#10B981",
            "code_bg": "#0F172A",
        }

    def render(self, text):
        for k, v in self.colors.items():
            print(f"  {k}: {v}")

theme = DarkTheme()
theme.render("Hello Dark World")

总结

暗黑主题通过绿色标签标题、暗色代码块、多色高亮块的组合，在白色背景上创造出清晰又专业的阅读体验。可直接用 theme: "dark" 在 config 中启用。
